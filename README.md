# Acrade-game
# How to play:
This game contain one player with much of enemy bugs .By using key arrows to cross the player between enemy without collision until reach the water if touch water get one point and back to original location and if collision with enemy the lives will be decreased then back to original location .Finally the player will won when 5poi.
